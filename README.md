
# Death's Light

A simple 2D Platformer based on the theme "Avoid the Light"



## Features

- Hide in the shadows to avoid the light
- Singleton patttern for SoundManager
- Implementation of patrolling enemies.


## Tech Stack

**Game Engine:** Unity-3D
